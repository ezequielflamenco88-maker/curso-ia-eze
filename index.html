Curso Profesional: IA y Generación de Ingresos

Este curso enseña a generar ingresos usando Inteligencia Artificial, desde la idea inicial hasta la publicación y monetización de cursos y aplicaciones educativas.

Introducción

Dirigido a personas que quieren formar parte del futuro digital y aprender a monetizar sus habilidades con IA. Al finalizar el curso, el alumno podrá aplicar IA para generar contenido, organizar cursos y crear ingresos de manera práctica y profesional.

MÓDULO 1 – Fundamentos de la Inteligencia Artificial aplicada a educación y emprendimiento digital

Objetivo: Entender qué es la IA, cómo funciona y cómo puede ayudarte a crear cursos y apps rentables.

Conceptos clave

  • IA generativa: textos, imágenes, videos, música, código.
  • Automatización: ahorro de tiempo y esfuerzo.
  • Creatividad asistida: generación de ideas, guiones y diseños.

Referencias académicas: Russell & Norvig (2016), Pedagogía de la Tecnología Educativa, UNESCO 2021.

Ejemplo práctico paso a paso

  1. Abrir ChatGPT u otra IA de texto.
  2. Escribir prompt: “Genera 10 ideas para un curso online sobre productividad usando IA”.
  3. Copiar las ideas en un documento.
  4. Crear carpetas img y videos para recursos multimedia.
  5. Vacíos para imágenes y videos: dejar cuadros de texto o comentarios indicando dónde se insertarán recursos.
Mini-ejercicio: Seleccionar 5 ideas y definir la principal. Guardar captura de pantalla de la IA mostrando las ideas.
Escenario de error: si la IA devuelve ideas repetidas, refinar el prompt con “Ideas originales y distintas para principiantes”.

Resultado esperado: Lista de ideas y selección de la idea principal.

MÓDULO 2 – Diseño de tu curso o app

Objetivo: Crear la estructura de tu curso o aplicación educativa paso a paso, asegurando coherencia pedagógica y facilidad de aprendizaje.

Explicación

Cada curso se organiza de lo simple a lo complejo. Se generarán módulos, lecciones y ejercicios con coherencia pedagógica. La IA ayuda a generar ideas iniciales y a optimizar creatividad y personalización.

Paso a paso práctico

  1. Abrir Google Docs o Word.
  2. Definir de 4 a 6 módulos principales.
  3. Para cada módulo, crear de 2 a 4 lecciones.
  4. Usar IA para generar títulos y resúmenes de lecciones con prompt: “Escribe títulos y resúmenes de lecciones sobre [tema elegido] para principiantes”.
  5. Guardar todo en carpetas ordenadas.
  6. Vacíos para imágenes y videos: dejar cuadros de texto o comentarios indicando dónde se insertarán recursos.
Ejercicio avanzado: Crear esquema completo del curso, incluyendo módulos, lecciones, objetivos de aprendizaje y competencias. Documentar rutas de carpetas y nombres de archivos multimedia.
Escenario de error: módulo demasiado extenso → dividir en submódulos; lecciones duplicadas → revisar coherencia.

Resultado esperado: Esquema completo con módulos, lecciones, objetivos de aprendizaje y espacios multimedia listos para agregar contenido visual.

MÓDULO 3 – Creación de contenido textual con IA

Objetivo: Redactar lecciones claras, didácticas y profesionales usando IA, garantizando comprensión y aplicabilidad para principiantes.

Paso a paso

  1. Abrir ChatGPT u otra IA de texto.
  2. Prompt de ejemplo: “Escribe una lección de 500 palabras sobre [tema], incluyendo ejemplos, preguntas de reflexión y desafíos prácticos”.
  3. Copiar el texto al documento principal del curso.
  4. Revisar claridad, coherencia, estilo académico y consistencia con objetivos de aprendizaje.
  5. Vacíos para insertar imágenes, diagramas y videos relacionados con la lección.
Ejercicio: Generar una lección por módulo. Crear 3 preguntas de reflexión o quiz por lección.
Escenario de error: IA genera contenido demasiado técnico → ajustar prompt a “nivel principiante”. Contenido incompleto → dividir la lección en secciones más pequeñas.

Evaluación formal

  • Rubrica: claridad 30%, relevancia 30%, ejercicios prácticos 20%, integración multimedia 20%.

MÓDULO 4 – Recursos visuales y multimedia

Objetivo: Crear imágenes, gráficos y videos educativos usando IA, optimizando la creatividad y personalización para tu curso.

Paso a paso

  1. Elegir herramienta de IA para imágenes (DALL·E, MidJourney, Canva).
  2. Prompt ejemplo para infografía: “Crea una infografía sobre cómo organizar un curso online con IA, colores suaves, estilo profesional”.

  6. Escenarios de error: imagen pixelada → aumentar resolución; video no reproduce → revisar codec/formato; IA genera contenido irrelevante → ajustar prompt.
Ejercicio práctico: Crear 3 imágenes educativas y 1 video de 1 minuto explicativo. Ajustar estilo, colores y tipografía según público objetivo.
Optimización de creatividad: combinar prompts, estilos visuales y variaciones de diseño para generar recursos únicos y profesionales.

MÓDULO 5 – Integración de contenido en HTML y publicación inicial

Objetivo: Montar tu curso en una página web funcional desde cero, incluyendo texto, multimedia y estructura profesional.

Paso 1: Crear archivo HTML desde cero

  1. Crear carpeta principal en el escritorio: MisCursosIA.
  2. Abrir Bloc de notas (Windows) o TextEdit (Mac).
  3. Guardar archivo como index.html, codificación UTF-8, dentro de MisCursosIA.
  4. Explicación: index.html es el archivo principal que los navegadores buscan al abrir un sitio.
  5. Tip: nombres exactos, sin espacios, minúsculas y extensión correcta (.html).

Paso 2: Estructura básica de HTML

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Curso de IA</title>
</head>
<body>
  <h1>Mi Curso de IA</h1>
  <h2>Módulo 1 – Fundamentos</h2>
  <p>Texto generado por IA...</p>
  <img src="img/infografia1.png" alt="Infografía módulo 1">
  <video controls>
    <source src="videos/video1.mp4" type="video/mp4">
  </video>
</body>
</html>

Explicación de cada etiqueta: <!DOCTYPE html> define tipo de documento, <html lang="es"> idioma, <head> meta info, <title> pestaña, <body> contenido visible, <h1>/<h2> títulos, <p> párrafos, <img> imágenes, <video controls> videos.

Vacío para imágenes y videos adicionales

Paso 3: Crear carpetas para recursos

  • Dentro de MisCursosIA: img para imágenes y videos para videos.
  • Tip: mantener nombres consistentes y extensiones correctas (.png, .jpg, .mp4).

Paso 4: Crear repositorio en GitHub

  1. Ir a github.com y crear cuenta si no existe.
  2. Click en New Repository.
  3. Nombre: CursoIA, público, marcar Initialize with README.
  4. Subir carpeta MisCursosIA: Add file → Upload files → Commit changes.
  5. Explicación: GitHub permite control de versiones y compartir tu curso online.

Paso 5: Activar GitHub Pages

  1. Ir a Settings → Pages.
  2. Fuente: seleccionar main branch.
  3. Carpeta raíz: /root.
  4. Guardar cambios.
  5. GitHub generará link público: https://tuusuario.github.io/CursoIA/.
  6. Explicación: GitHub Pages permite publicar tu curso como sitio web sin servidor externo.
Ejercicio práctico: Crear index.html, carpetas img y videos, subir archivos, activar Pages, probar link público.
Escenarios de error: archivos no cargan → revisar rutas/nombres exactos; video no reproduce → verificar formato/codec.
Resultado esperado: Curso online funcionando con texto, imágenes y video.

MÓDULO 6 – Evaluaciones y ejercicios prácticos

Objetivo: Crear cuestionarios automáticos y desafíos prácticos que midan el aprendizaje del alumno de manera profesional.

Paso a paso:

  1. Generar preguntas con IA: multiple choice, verdadero/falso y abiertas.
  2. Integrar cuestionarios en Google Forms o directamente en HTML con formularios.
  3. Revisar coherencia de las preguntas con cada lección.
  4. Vacíos para imágenes o videos dentro de preguntas interactivas.
  5. Escenarios de error: pregunta ambigua → revisar claridad; enlaces rotos → actualizar URL.

Ejercicios avanzados:

  • Crear 5 preguntas por módulo con distintos niveles de dificultad (básico, intermedio, avanzado).
  • Integrar evaluación formal tipo universidad.

Rúbrica de evaluación:

  • Pertinencia del contenido: 40%
  • Claridad en redacción: 30%
  • Dificultad y resolución: 30%

Ejercicio práctico:

Módulo 1 – Teoría pedagógica y creatividad

Módulo 1 – Teoría pedagógica y optimización de creatividad

Objetivo:

Comprender cómo se aplican principios pedagógicos para el aprendizaje con IA y cómo optimizar la creatividad y personalización de los cursos.

Teoría pedagógica aplicada:

Este módulo introduce principios de diseño instruccional y aprendizaje significativo:

  • Constructivismo: el alumno construye conocimiento mediante la interacción con IA y ejercicios prácticos.
  • Aprendizaje basado en problemas: escenarios reales de aplicación de IA para generar ingresos.
  • Gamificación y motivación: uso de retos, puntuaciones y ejercicios para mantener la atención y el compromiso.

Referencias académicas:

  • Merrill, M. D. (2002). First principles of instruction.
  • UNESCO. (2021). Pedagogical Guidelines for Technology Integration in Education.
  • Luckin, R. et al. (2016). Intelligence Unleashed: AI in Education.

Optimización de creatividad y personalización:

  1. Analizar intereses del alumno y nivel de conocimiento.
  2. Diseñar prompts que generen contenido adaptado a distintos estilos de aprendizaje.
  3. Crear múltiples versiones de lecciones para comparación y mejora.
  4. Vacíos para ejemplos de contenido generado con distintas estrategias creativas.

Discusión crítica sobre IA:

Es importante que los alumnos comprendan tanto el potencial como las limitaciones de la IA:

  • IA no sustituye al docente ni la pedagogía, es una herramienta complementaria.
  • El contenido generado debe revisarse para evitar errores, sesgos o falta de coherencia.
  • Reflexionar sobre ética y privacidad en el uso de datos para IA.
“La inteligencia artificial es un amplificador de creatividad, no un reemplazo del pensamiento crítico.” – Luckin et al., 2016

Módulo 5 – Integración de contenido en HTML y publicación inicial

Módulo 5 – Integración de contenido en HTML y publicación inicial

Objetivo:

Montar el curso en una página web funcional desde cero, con integración de imágenes, videos y recursos interactivos.

Paso 1: Crear la carpeta y archivo HTML

  1. Crear carpeta MisCursosIA en el escritorio.
  2. Abrir Bloc de notas (Windows): Inicio → Buscar “Bloc de notas” → Enter.
  3. Guardar archivo: Archivo → Guardar como → Nombre: index.html → Tipo: “Todos los archivos” → Codificación: UTF-8 → Ubicación: dentro de MisCursosIA.
  4. Explicación: index.html es el archivo principal que los navegadores buscan al abrir un sitio web.
  5. Tip: usar nombres exactos, sin espacios, respetar minúsculas y extensión .html.

Paso 2: Estructura básica de HTML

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Curso de IA</title>
</head>
<body>
  <h1>Mi Curso de IA</h1>
  <h2>Módulo 1 – Fundamentos</h2>
  <p>Texto generado por IA...</p>
  <div class="placeholder">Imagen o infografía aquí</div>
  <div class="placeholder">Video aquí</div>
</body>
</html>

Paso 3: Crear carpetas para recursos

  • Dentro de MisCursosIA, crear carpeta img para imágenes y videos para videos.
  • Tip: mantener nombres consistentes y extensiones correctas (.png, .jpg, .mp4).

Paso 4: Crear repositorio en GitHub

  1. Ir a github.com y crear cuenta si no tienes.
  2. Click en “New Repository” → Nombre: CursoIA → Público → Marcar “Initialize with README”.
  3. Subir carpeta MisCursosIA: “Add file” → “Upload files” → Commit changes.
  4. Explicación: GitHub es un sistema de control de versiones que permite alojar y compartir tu curso online.

Paso 5: Activar GitHub Pages

  1. Ir a Settings → Pages.
  2. Fuente: seleccionar main branch.
  3. Carpeta raíz: /root.
  4. Click Save.
  5. GitHub generará un link público: https://tuusuario.github.io/CursoIA/
  6. Explicación: GitHub Pages permite publicar tu curso como sitio web sin necesidad de servidor externo.

Ejercicios prácticos:

  • Crear archivo index.html, carpetas img y videos, y subir archivos de prueba.
  • Activar GitHub Pages y abrir link público.

Escenarios de error frecuentes:

  • Archivos no cargan → revisar rutas y nombres exactos.
  • Videos no reproducen → verificar formato y codec.

Resultado esperado:

Curso online funcionando con texto, imágenes y video correctamente integrados.
